# Instagram Social Media Analytics

## Project Overview
This project analyzes user data from Instagram platform to help a marketing team improve user engagement, retention and acquisition. It focuses on identifying peak activity times, top influencers, and content trends.

## Repository Contents
* **ig_clone.sql**: The core database script containing the schema (CREATE TABLE) and initial data population.
* **Social_Media_Analysis_p.sql**: SQL script containing queries for data cleaning, user ranking, and engagement analysis.
* **Social Media Analysis - Tasks.docx**: Detailed report with technical steps, query results, and data interpretations.
* **Social_Media_Analysis.pptx**: Executive summary presentation featuring key visualizations and strategic recommendations.

## Key Insights
* **Peak Times:** Users are most active on Tuesdays and Thursdays between 4 PM and 7 PM.
* **User Stats:** Identified that 26% of users have zero posts, suggesting a major opportunity for re-engagement campaigns.
* **Top Content:** "Party," "Food," and "Fashion" are the most used hashtags, while "Smile" has the highest engagement rate.
* **Influencers:** Ranked top candidates for brand partnerships using a weighted score (60% engagement / 40% follower count).

## How to Use
1. **Database Setup:** Run `ig_clone.sql` first to create the database structure and load the data.
2. **Run Analysis:** Execute `Social_Media_Analysis_p.sql` to generate the insights and analytics used in the report.
3. **Review Results:** Refer to the `.docx` for methodology and the `.pptx` for the final stakeholder presentation.

## Tools Used
* **SQL:** Database management and data extraction.
* **Excel:** Data visualization and Pivot Tables.
* **PowerPoint:** Stakeholder reporting and storytelling.
