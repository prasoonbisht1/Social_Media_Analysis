# Instagram Social Media Analytics

## Project Overview
This project analyzes user data from an Instagram-like platform to help a marketing team improve user engagement and retention. It focuses on identifying peak activity times, top influencers, and content trends.

## Repository Contents
* **Social_Media_Analysis_p.sql**: SQL script containing queries for data cleaning, user ranking, and engagement analysis.
* **Social Media Analysis - Tasks.docx**: Detailed report with technical steps, query results, and data interpretations.
* **Social_Media_Analysis.pptx**: Executive summary presentation featuring key visualizations and strategic recommendations.

## Key Insights
* **Peak Times:** Users are most active on Tuesdays and Thursdays between 4 PM and 7 PM.
* **Top Content:** "Party," "Food," and "Fashion" are the most used hashtags, while "Smile" has the highest engagement rate.
* **User Stats:** Identified that 26% of users have zero posts, presenting a major opportunity for re-engagement campaigns.
* **Influencers:** Ranked top candidates for brand partnerships using a weighted score of engagement and follower count.

## How to Use
1. **SQL:** Run the `.sql` file in your database to see how the data was extracted and segmented.
2. **Report:** Read the `.docx` file for a deep dive into the business questions and technical methodology.
3. **Slides:** Open the `.pptx` file for a quick visual overview of the final findings and marketing advice.
